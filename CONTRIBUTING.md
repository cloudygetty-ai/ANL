# Contributing to ANL

This is a private repository. Contributions are by invitation only.

## Setup

1. Clone the repo
2. `npm install`
3. Copy `.env.example` to `.env`
4. Set up Firebase (see `FIREBASE_SETUP.md`)
5. Set up PostgreSQL with PostGIS
6. `npm start`

## Code Standards

- React Native with JSX
- Express backend in `/backend`
- Conventional commits
