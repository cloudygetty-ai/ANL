Built for the night

Every feature is designed around how nightlife actually works.

📍

Live map

Nearby people appear as distinctive pins — shape and color reflect gender identity. No one is a generic dot.

🌊

NightPulse

A real-time heatmap of activity intensity across your city's zones. Find where the night is actually happening before you go.

✨

Vibe signals

Send a lightweight wave or spark — no pressure of a full message. Mutual vibes create a match automatically.

💬

Real-time chat

Instant delivery, read receipts, and group channels. Messages arrive as fast as the network allows — no polling.

🎥

Video rooms

Start a video or audio call directly in the app via LiveKit. No third-party apps, no links to copy.

🔄

Always on

An adaptive background loop persists state across OS kills and heals from errors automatically — without user input.
